# datascienceQCM

A Streamlit-based multiple choice quiz (QCM) application to test your Data Science knowledge.

## 🚀 Live App

👉 [https://datascienceqcm.streamlit.app/](https://datascienceqcm.streamlit.app/)

## Features

- Multiple choice questions on Data Science topics
- 15-second timer per question (random answer assigned if time runs out)
- Score tracking and results saved to CSV
- Requires a valid `@dauphine.psl.eu` email to participate

## Running Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
